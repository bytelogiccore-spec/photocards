# 포토카드 메이커

Supabase + 바닐라 JavaScript로 만든 포토카드 제작 앱

## 배포 방법

### Vercel 배포

1. GitHub 저장소 생성
2. 이 폴더 내용 푸시
3. Vercel에서 Import
4. 자동 배포 완료

### 로컬 실행

```bash
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속

## 환경 변수

필요 없음 (API 키가 코드에 포함되어 있음)

## 기능

- ✅ 이메일 로그인/회원가입
- ✅ 4가지 템플릿 (클래식, 모던, 빈티지, 미니멀)
- ✅ 이미지 업로드
- ✅ 텍스트 추가 (색상 변경 가능)
- ✅ 스티커 추가
- ✅ 포토카드 저장 (Supabase)
- ✅ 다운로드
- ✅ 내 작품 관리
